# Xavious - 횡스크롤 슈팅 게임

고전 게임 제비우스(Xevious)를 모티브로 한 횡스크롤 슈팅 게임입니다.

## 게임 특징

- 횡스크롤 슈팅 액션
- 플레이어 비행기 조작
- 다양한 적 비행기
- 폭발 효과 및 파티클 시스템
- 점수 시스템

## 조작법

- **← → ↑ ↓ 화살표 키**: 플레이어 이동
- **스페이스바**: 총알 발사
- **R 키**: 게임 재시작

## 실행 방법

브라우저에서 `index.html` 파일을 열면 바로 플레이할 수 있습니다.

## 게임 목표

- 적 비행기를 파괴하여 점수를 획득하세요
- 적과 충돌하면 생명이 감소합니다
- 생명이 0이 되면 게임 오버입니다
- 빨간색 적은 더 빠르지만 더 높은 점수를 줍니다

## 기술 스택

- HTML5 Canvas
- Vanilla JavaScript
- CSS3

즐거운 플레이 되세요!
