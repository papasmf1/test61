# Super Mario - 횡 스크롤 게임

횡 스크롤 방식의 수퍼마리오 스타일 웹 게임입니다.

## 게임 특징

- 플레이어 캐릭터 (마리오 스타일)
- 좌우 이동 및 점프
- 중력 및 물리 시스템
- 플랫폼 이동
- 적 캐릭터와의 상호작용
- 코인 수집
- 카메라 스크롤
- 생명 시스템

## 조작법

- **← (왼쪽 화살표)** 또는 **A**: 왼쪽으로 이동
- **→ (오른쪽 화살표)** 또는 **D**: 오른쪽으로 이동
- **스페이스바** 또는 **↑ (위 화살표)**: 점프

## 게임 규칙

- 코인을 수집하면 100점 획득
- 적을 위에서 밟으면 200점 획득 (적 제거)
- 적과 충돌하면 생명 감소 및 리스폰
- 생명이 0이 되면 게임 오버

## 실행 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. 게임 시작!

## 게임 요소

- **플랫폼**: 갈색 블록으로 표시
- **코인**: 금색 동전 (수집 가능)
- **적**: 갈색 블록 형태의 적
- **플레이어**: 빨간색 마리오 스타일 캐릭터
